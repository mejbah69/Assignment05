Question-1: What are the different ways to select an element in the DOM?

Different ways are usage of methods like getElementById (by ID), getElementsByClassName (by class name), getElementsByTagName (by tag name), querySelector (using CSS selectors to select a single element), and querySelectorAll (using CSS selectors to select multiple elements).

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML includes HTML, innerText shows only visible text, and textContent includes all text (hidden + visible) without HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and propagates upward through its parent elements in the DOM.
